# Web AR ดอกกระเจียวแดง

โปรเจกต์ Static Web AR พร้อมนำขึ้น Vercel ใช้วิดีโอ `assets/siam-tulip-loop.mp4` และมาร์กเกอร์ Hiro วางบนพื้น

## วิธีทดสอบเร็วที่สุด

1. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub หรือแตก ZIP แล้วลากเข้า Vercel
2. เปิด URL ที่ได้บนมือถือผ่าน HTTPS
3. กด **เริ่มเปิดกล้อง AR** และอนุญาตกล้อง
4. เปิด `marker.html` บนอุปกรณ์อีกเครื่อง หรือพิมพ์ `assets/hiro-marker-print-a4.png`
5. วางมาร์กเกอร์ราบบนพื้น หันกล้องเข้าหาในมุมประมาณ 30–60 องศา
6. เมื่อพบมาร์กเกอร์ ดอกกระเจียวจะเริ่มเล่นจากต้นคลิปและเล่นวน

## ไฟล์สำคัญ

- `index.html` หน้า Web AR
- `marker.html` หน้าดู/พิมพ์มาร์กเกอร์
- `qr.html` เครื่องมือสร้าง QR หลังทราบ URL จริง
- `assets/siam-tulip-loop.mp4` วิดีโอ 24 วินาที
- `assets/hiro-marker-print-a4.png` แผ่นพิมพ์ A4

## ปรับขนาดและตำแหน่งดอกไม้

ใน `index.html` ค้นหา `id="flowerPlane"`

- `width="1.62"` และ `height="0.91"` คือขนาดภาพ
- `position="0 0.52 0"` ค่าแกน Y คือความสูงจากพื้น

## หมายเหตุ

- ต้องเปิดผ่าน HTTPS หรือ localhost จึงใช้กล้องได้
- มาร์กเกอร์ควรเรียบ ไม่สะท้อนแสง และมีขนาดอย่างน้อย 15–20 ซม.
- วิดีโอมีพื้นหลังสีน้ำเงิน ระบบลบสีด้วย GPU shader แบบเรียลไทม์
- หากขอบใบมีสีน้ำเงิน ให้เพิ่ม `similarity` เล็กน้อย เช่น 0.070 หรือเพิ่ม `smoothness` เป็น 0.130
